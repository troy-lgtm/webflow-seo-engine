# Webflow Rendering Guide

Use this to render diagram + scorecard visuals generated by the SEO engine.

## 1) Mermaid diagram rendering

Add this once in page settings (before `</body>`):

```html
<script type="module">
  import mermaid from 'https://cdn.jsdelivr.net/npm/mermaid@11/dist/mermaid.esm.min.mjs';
  mermaid.initialize({ startOnLoad: true, theme: 'default' });
</script>
```

Place the `diagram_mermaid` field inside:

```html
<pre class="mermaid">{{wf {&quot;path&quot;:&quot;diagram_mermaid&quot;,&quot;type&quot;:&quot;PlainText&quot;} }}</pre>
```

## 2) McKinsey-style visual cards

Option A:
- Use `scripts/export_visual_embed.sh` and paste resulting HTML into an Embed component.

Option B:
- Parse `visual_cards` JSON with custom code and render cards with your own classes.

## 3) Safety

- Keep diagrams informational (no fabricated metrics).
- Keep visuals readable on mobile.
